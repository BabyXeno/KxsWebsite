Documentation

Getting Started

Welcome to the KxsClient documentation. Follow these steps to begin:

  1. Download a script loader extension like TamperMonkey from your browser's extension store
  2. Visit our download page and get the latest KxsClient script
  3. Open TamperMonkey and create a new script, then paste the KxsClient code
  4. Save the script and configure your preferences in the settings panel
  5. Join a game and enjoy your enhanced gaming experience!

Configuration Guide

KxsClient offers extensive customization options to enhance your gameplay:

  1. Press Alt+K in-game to open the settings panel
  2. Adjust HUD elements through the drag-and-drop interface
  3. Customize colors, transparency and size of all UI components
  4. Save multiple profiles for different game modes
  5. Import and export settings to share with friends

Frequently Asked Questions

Find answers to common questions about KxsClient:

  1. Is KxsClient safe to use?
    Yes, KxsClient is designed with security in mind and doesn't modify game files directly.
  2. Will it work on all browsers?
    KxsClient is compatible with Chrome, Firefox, Edge, and other major browsers that support userscripts.
  3. Is KxsClient free?
    Yes, the core features are completely free. We may offer premium features in the future.
  4. How do I report bugs?
    Join our Discord server and post in the #bug-reports channel.
